# Dados-Mestrado

Repositório da dissertação de mestrado contendo bases de dados, scripts de análise e gráficos da pesquisa empírica.

## 📁 Estrutura do Repositório
```
Dados-Mestrado/
├── Gráficos Tableau/     # Visualizações e dashboards criados no Tableau
├── Programas Python/     # Scripts de análise e tratamento de dados
├── Tabelas Amostrais/    # Bases de dados e amostras utilizadas na pesquisa
└── README.md
```
## 📊 Conteúdo

- **Gráficos Tableau** — Visualizações gráficas geradas a partir dos dados da pesquisa, utilizadas na dissertação.
- **Programas Python** — Scripts desenvolvidos para coleta, limpeza, tratamento e análise estatística dos dados.
- **Tabelas Amostrais** — Planilhas Excel com as bases de dados e amostras utilizadas na pesquisa empírica.

## 🛠️ Tecnologias Utilizadas

- Python
- Tableau
- Microsoft Excel

## 📌 Sobre

Este repositório reúne os artefatos de dados produzidos durante a pesquisa de mestrado, organizados para fins de reprodutibilidade e transparência científica.
